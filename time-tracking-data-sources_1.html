★ label is the key reporting dimension

Label is a composite of client + project (routes 1 and 2) or client + "ops" (route 3), computed at write time. It carries full context in a single column — the dashboard needs no joins, no filters, one GROUP BY.

SELECT label, SUM(hours) AS total_hours
FROM time_tracking
GROUP BY label
